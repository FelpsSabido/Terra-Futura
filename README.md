Sobre o projeto:

O TerraFutura é um projeto desenvolvido para o Programa Agrinho com o objetivo de apresentar como a tecnologia, a sustentabilidade e a inovação podem contribuir para um futuro mais equilibrado entre o desenvolvimento humano e a preservação ambiental.

O projeto utiliza recursos interativos para permitir que os usuários explorem informações sobre os estados brasileiros e conheçam iniciativas sustentáveis aplicadas ao agronegócio.

Objetivos
- Demonstrar a importância da sustentabilidade no campo.- Apresentar tecnologias que contribuem para uma produção mais eficiente.- Incentivar a conscientização ambiental.- Mostrar como inovação e preservação podem caminhar juntas.

Funcionalidades
 Mapa Interativo do Brasil
- Todos os estados são clicáveis.- Exibição dinâmica de informações de cada estado.- Destaque visual ao selecionar um estado.
 Agricultura Sustentável
- Informações sobre as principais produções agrícolas.- Destaque para práticas sustentáveis utilizadas em cada região.
 Tecnologias para o Futuro
- Agricultura de precisão.- Internet das Coisas (IoT).- Energia solar.- Monitoramento por drones.

Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript
- SVG Interativo

Como Executar
1. Baixe ou clone o projeto.2. Abra a pasta no Visual Studio Code.3. Execute utilizando a extensão Live Server.4. Abra o navegador para visualizar a aplicação.

Público-Alvo
Estudantes, professores e visitantes interessados em sustentabilidade, inovação tecnológica e desenvolvimento do agronegócio brasileiro.
